# Get started

Asko Storybook is a reusable component library that helps Storybook contributors build UIs faster. The goal is to make building durable UIs more productive and satisfying.

## Install

Add Asko Storybook package to your project.

`npm install --save @asko-storybook/{PACKAGE_NAME}@PACKAGE_VERSION`

### **Use**

Import components you want into your UI

`import BaseButton from ‘@asko-storybook/button/BaseButton’;`

and use them like so

```

<template>
  <div class="someclass">
    <BaseButton label="Button" size="medium" backgroundColor="blue" @onClick="greet($event)"/>
    OR
    <base-button label="Button" size="medium" backgroundColor="lightgreen" @onClick="greet($event)"></base-button>
  </div>
</template>

<script>
  import BaseButton from "@asko-storybook/button/BaseButton";

  export default {
    name: "ButtonExample", 
    components: {
      BaseButton,
    },
    methods: {
      getBaseURI: function (event) {
        alert(event.target.baseURI)
      }
    }
  };
  </script>
```
