# Get started

Asko Storybook is a reusable component library that helps Storybook contributors build UIs faster. The goal is to make building durable UIs more productive and satisfying.

## Install

Add Asko Storybook package to your project.

`npm install --save @storybook/asko-storybook`

### **Use**

Import components you want into your UI

`import { Button } from ‘@storybook/asko-storybook’;`

and use them like so

```
  const example = () => (
    <div>
      <Button appearance="primary" onClick={() => 0}>Do something</Button>
    </div>
  )
```
